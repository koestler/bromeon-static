C++ library issues

If you encounter a bug or have a specific question concerning one of my libraries, please send me a minimal and complete example code which clearly describes the problem. That means, try to avoid code that is specific to your project or other libraries (except SFML), and make sure that the code is compilable - at least apart from your issue.

If you have general design questions or feature requests, I will like to hear them! Note that the feature should be in scope of the library, and there must exist meaningful use cases to make it into a future version.
